# Contributing to Nexus Launcher

Thank you for your interest in improving Nexus Launcher! To keep the project maintainable and high-quality, please follow these guidelines.

## Where to Start

**Not sure what to work on?** We have numerous `TODO` comments scattered throughout the codebase representing pending tasks, optimizations, and known minor issues that urgently need attention. Searching for `TODO` in your editor is a fantastic way to find your first contribution and jump right in!

## How to Contribute

1.  **Report Bugs**: Open an issue describing the bug, your environment, and steps to reproduce it.
2.  **Suggest Features**: Open an issue to discuss new ideas before implementation.
3.  **Tackle TODOs**: Search the source code for `TODO` to resolve pending tasks and submit your fixes.
4.  **Submit Pull Requests**:
    * Fork the repository and create your branch from `main`.
    * Ensure your code follows the existing style.
    * Write clear commit messages.
    * Open a PR with a description of your changes.

## Development Standards

* **Rust Version**: Use the latest stable Rust.
* **Formatting**: Always run `cargo fmt` before committing.
* **Testing**: Add tests for new features and ensure all existing tests pass with `cargo test`.

## Getting Help

If you have questions, feel free to open an issue for discussion or reach out in our community chat.
